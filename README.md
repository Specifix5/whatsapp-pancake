# Whatsapp Pancake Bot
Utility & Reminder Whatsapp Bot made in WWeb.JS
### Commands
Type `!help` to your Whatsapp Bot for commands.

# Setting the bot up
* Install all dependencies:
```bash
npm install
```
* All configs in an `.env` file. Example of `.env` file:
```.env
DEBUG_MODE=true
TIMEZONE=Etc/UTC
```
* Deploy with node
```bash
npm run dev
```
